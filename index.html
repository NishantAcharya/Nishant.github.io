Projects and Reports

  • Transformers for logical reasoning :
    This study takes a look at how can we represent and generate the transformer block weights, such that logical connections can be represented in a transformer.
  • Emotion Based ADL Detection :
    In this paper, we propose a system named Emotion Based ADL Detection (EBADL) over non-invasive sensors. The project utilizes emotional context to build a activity detection system for a spectrum of emotions. We achieve this using transfer learning over a similar dataset and emphasize that it is possible to use ambient sensing to provide long term care to the elderly.
  • Analysis of Multimodal Timeseries Fusion using Boltzmann Machines :
    We explored the most popular ways of fusing multimodal data in a distributed way using boltzmann machines and provided an experimental algorithm to apply this in the MPC framework.
  • Transformers for Classifying Multivariate Timeseries Data :
    In this work, we investigate a popular classification framework that uses a transformer encoder for unsupervised representation learning of multivariate time series to train on supervised learning tasks, and examine whether a similar accuracy mentioned in the study can be achieved for low dimensional MTS data.
  • Intermittent Network Disaster Management Application: Kapok :
    Inspired by RNRNs resolve in hurricane Maria and the nurses' resourcability in intermittent network conidtions, me and a team of developers under Dr. Leah Wing built a disaster management application for intermittent network conditions, keeping the original first responders to Maria in the loop.
  • RISC Vector Extension :
    Defined a RISC based ISA from scratch for a 64 bit system and wrote programs in binary to demonstrate the vector extension on a matrix multiplication benchmark.
  • Job Search Web Application :
    Created an AWS based job search web application in a multi-team collaboration.
  • RDT 3.0 Simulator :
    In this project, I simulated a RDT 3.0 server and demonstrated the workings of TCP at application layer and calculated the utilization rate of the link for scenarios like, defined congestion rate, low traffic,unreliable data transfer.
